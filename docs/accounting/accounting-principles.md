# Accounting Principles

Financial implementation is not part of Step 1.

When accounting work begins, the implementation must follow these non-negotiable controls:

- Money must use decimal fields.
- Every posted financial transaction must create balanced journal entries.
- Posted journal entries are immutable.
- Corrections must use reversal and reposting.
- Closed accounting periods and hard-closed academic years must block posting.
- Maker-checker approval is mandatory for financial transactions.
- The user who creates a financial transaction cannot approve it.
- The general ledger is the financial source of truth.
- Advance student payments are liabilities until applied.
